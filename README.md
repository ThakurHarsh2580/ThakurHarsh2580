## Featured Projects

### ResuMind — AI Resume Analyzer
> *React.js · React Router · Tailwind CSS · Puter.js AI API* &nbsp; `2026`

An AI-powered Applicant Tracking System that enables employers to post job listings, upload candidate resumes, and receive automated fit scores — with **zero backend infrastructure**.

- Integrated AI API for automated resume-to-job matching and structured feedback generation
- Built a PDF-to-image ingestion pipeline for real-time candidate scoring
- Third-party auth and cloud file storage via Puter API — secure, zero-config deployment

---

### Employee Task Management System
> *React.js · React Router · Context API · Tailwind CSS* &nbsp; `2025` &nbsp; · &nbsp; [**Live Demo**](https://task-management-system-eight-tawny.vercel.app/)

A role-based task management platform with dedicated **Admin** and **Employee** dashboards.

- Centralized state via React Context API (AuthContext + TaskContext) — no prop drilling
- Full task lifecycle: create, assign, prioritize (High/Medium/Low), and track status
- Color-coded priority system, reusable modal/spinner components, persistent sessions

---

### LuxeCart — E-Commerce Application
> *React.js · React Router · Context API · Tailwind CSS* &nbsp; `2026` &nbsp; · &nbsp; [**Live Demo**](https://luxe-cart-ecommerce-cyan.vercel.app/)

A fully responsive e-commerce platform with a smooth, modern shopping experience.

- Three decoupled Context stores (Product, Cart, Wishlist) for scalable state management
- Category browsing, keyword search, image galleries, ratings, reviews, and stock indicators
- Persistent cart and wishlist via browser storage, deployed on Vercel with Vite build
